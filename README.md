# VFBuild
Стартовый шаблон для верстки

## Используемые технологии
* [Vite](https://vitejs.dev/)
* [БЭМ](https://ru.bem.info/)
* [SCSS](http://sass-lang.com/)

## Структура папок
"+" - помечены файлы и папки доступные для использования

"-" - помечены файлы и папки НЕ доступные для использования

```
|+ dist/                            - папка с prod-файлами
|- node_modules/                    - папка с node-модулями
|- pages/                           - папка с страницами
    |+ about.html                   - Страница "О нас"
|- src/                             - папка с исходниками
    |- assets/                      - папка с ресурсами
        |- fonts/                   - папка с шрифтами
        |- images/                  - папка с изображениями
        |- sprites/                 - папка с спрайтами
    |- scripts/                     - папка со скриптами
        |-animation                 - папка с анимациями
        |-components                - папка с компонентами
        |-helpers                   - папка с хелперами
        |-pages                     - папка с страницами
            |+ home.js              - скрипт главной страницы
        |+ main.js                  - Общая логика для всех страниц
    |- styles/                      - папка со стилями
        |- animation/               - папка с анимациями
        |- components/              - папка с компонентами
        |- mixins/                  - папка с миксинами
        |- pages/                   - папка с страницами
            |+ home.scss            - стили главной страницы
        |+ _fonts.scss              - подключение шрифтов
        |+ _typography.scss         - Стили для текста
        |+ _variables.scss          - Переменные
        |+ main.scss                - Корневой файл стилей           
|- README.md                        - файл readme
|- eslintrc.js                      - файл конфигурации EsLint
|- prettierrc.js                    - файл конфигурации Prettier
|- package.json                     - файл конфигурации проекта npm
|- package-lock.json                - файл конфигурации проекта npm
|+ .gitignore                       - файл с исключениями git
|+ .vite.config.js                  - файл конфигурации Vite
```
